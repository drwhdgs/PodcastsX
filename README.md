# PodcastsX
Fixes Podcasts on iOS 6!

This tweak restores functionality to the native Podcasts app on iOS 6 by proxying TLS connections through mbedTLS to support modern ciphers and API endpoints.

## Features
- Fixes subscription syncing (Bookkeeper).
- Fixes podcast downloads and streaming.
- Redirects legacy endpoints to modern Apple APIs.
